### 项目说明
这是一个模拟的博客系统，旨在练习spring boot等知识点的应用，以java后端为主，本项目希望努力做成一个方便本地使用的文件和笔记管理系统，包含单不限于博客文章、pdf书籍阅读、资料管理等功能</br>

### 预计功能说明
1. 实现类似博客的文章访问功能，如文章标题、内容(基本实现)
2. 实现文章统计功能，如字数、时间、分类
3. 实现pdf书籍阅读功能
4. 实现资料上传和下载、资料列表功能
5. 实现基础的登录注册功能及权限控制（基本实现）
6. 根据不同用户权限展现不同页面内容（基本实现）
7. 实现推送功能
8. 实现评论邮件提醒功能（基本实现）
9. 日志记录（基本实现）
